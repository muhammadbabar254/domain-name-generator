# 🌐 Domain Name Generator

A modern and responsive web application that helps users generate creative and professional domain name ideas for startups, businesses, agencies, blogs, SaaS products, and personal brands.

---

# ✨ Features

* Generate creative domain name ideas
* Supports multiple domain extensions
* Business name based suggestions
* Copy domain name with one click
* Clean and responsive interface
* Fast and lightweight
* Duplicate-free results
* Modern SaaS-style design

---

# 🚀 Supported Extensions

* .com
* .net
* .org
* .io
* .co
* .app
* .dev
* .ai
* .tech
* .online
* .site
* .store
